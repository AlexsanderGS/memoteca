# 🧠 Memoteca

Projeto desenvolvido durante a **Carreira Desenvolvedor Front-End React** da Alura, no curso:

📚 **JavaScript: implementando CRUD com requisições HTTP**

O objetivo do projeto foi praticar a comunicação entre o Front-End e uma API, implementando operações completas de **CRUD** utilizando JavaScript.

---

## 📖 Sobre o projeto

A **Memoteca** é uma aplicação web para cadastro de **mensagens e citações**, onde o usuário pode:

- Criar novas mensagens
- Visualizar citações cadastradas
- Editar conteúdos existentes
- Excluir mensagens

Cada registro contém:

- ✍️ Conteúdo da mensagem
- 👤 Autor da citação

O projeto simula um fluxo real de aplicações modernas, consumindo dados de uma API através de requisições HTTP.

---

## 🚀 Funcionalidades

✅ Criar mensagens (Create)  
✅ Listar mensagens cadastradas (Read)  
✅ Editar mensagens existentes (Update)  
✅ Excluir mensagens (Delete)  
✅ Integração com API fake  
✅ Atualização dinâmica da interface

---

## 🌐 API Fake (JSON Server)

Para simular um backend real, foi utilizado o **JSON Server**, permitindo criar rapidamente uma API REST fake.

Foram utilizadas requisições HTTP:

- `GET` → Buscar mensagens
- `POST` → Criar nova mensagem
- `PUT` → Atualizar mensagem
- `DELETE` → Remover mensagem
